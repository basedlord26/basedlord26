## What I'm Building

### 🔨 Dating App MVP *(in progress)*
A dating app for Singapore built around a different premise: before matching people, understand what the community actually wants from dating. The app has an opinion-gathering and feedback layer built in — so the product evolves based on real user sentiment, not assumptions.

Most dating apps were built for Western markets and ported here. This one starts in Singapore.

**Status:** MVP in development
